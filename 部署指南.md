# 部署到 GitHub Pages　逐步指南

預計時間：約 5～10 分鐘（含 git 上傳 125 MB 的時間）。

## 一、先在 GitHub 建立 repo

1. 開啟瀏覽器到 <https://github.com/new>
2. **Owner**：silentcountry0109-hash
3. **Repository name**：`cap-pastpapers`
4. **Public**（必須選 Public，免費版的 GitHub Pages 才能用）
5. 不要勾選「Add a README」、「.gitignore」、「license」（保持空 repo）
6. 按下 **Create repository**

## 二、確認電腦已裝 git

打開命令提示字元（Windows：開始選單搜尋「cmd」），輸入：
```
git --version
```
若出現版本號（例如 `git version 2.42.0`）就是已安裝。

若沒裝，先到 <https://git-scm.com/download/win> 下載安裝。安裝過程一路 Next 即可。

## 三、執行部署腳本

### 方法 A：雙擊 deploy.bat（最簡單）

直接在檔案總管雙擊 `deploy.bat`。

第一次 push 時會跳出 GitHub 登入視窗，用瀏覽器登入即可（會自動記住憑證）。

### 方法 B：手動逐步（若 .bat 失敗）

打開命令提示字元，輸入：
```
cd "C:\Users\silen\Desktop\會考歷屆試題\_成品\教用網頁好讀版"

git init -b main
git add .
git -c user.email="silentcountry0109@gmail.com" -c user.name="silentcountry0109-hash" commit -m "Initial deploy"
git remote add origin https://github.com/silentcountry0109-hash/cap-pastpapers.git
git push -u origin main
```

第一次 push 會要你登入 GitHub。

## 四、開啟 GitHub Pages

push 完成後：

1. 開啟 <https://github.com/silentcountry0109-hash/cap-pastpapers/settings/pages>
2. **Source** 選擇 `Deploy from a branch`
3. **Branch** 選擇 `main` / `/ (root)`
4. 按 **Save**

等 1～2 分鐘，網址會出現在頁面頂端：

```
https://silentcountry0109-hash.github.io/cap-pastpapers/
```

## 五、產生 QR Code 並嵌回學用 PDF

部署完成、網址確定可用之後，回到「工具腳本」資料夾執行：

```
cd "C:\Users\silen\Desktop\會考歷屆試題\_成品\工具腳本"
python embed_qr.py https://silentcountry0109-hash.github.io/cap-pastpapers/
```

執行完會在 `outputs/dist_qr/` 產出三個含 QR 的學用 PDF。

## 常見問題

**Q. push 時要求登入但瀏覽器沒跳出來**　
A. 先在命令列執行 `git config --global credential.helper manager`，再 push 一次。

**Q. push 速度慢**　
A. 125 MB 在一般網速約 2~5 分鐘。中途進度條停在 99% 是 GitHub server-side 處理 large files，等一下即可。

**Q. Pages 顯示 404**　
A. 第一次 build 需要約 1~2 分鐘。若超過 5 分鐘，到 <https://github.com/silentcountry0109-hash/cap-pastpapers/actions> 看 build 是否失敗。

**Q. 之後想更新內容**
A. 改完檔案後在同資料夾執行：
```
git add .
git commit -m "更新內容"
git push
```
